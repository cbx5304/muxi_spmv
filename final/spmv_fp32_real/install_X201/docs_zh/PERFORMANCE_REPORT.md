# SPMV FP32 库性能测试报告 (Mars X201版本)

## 概要

`spmv_fp32` 库在 **Mars X201** GPU上的性能验证结果。

**关键结果**:
- 内核带宽: **1111 GB/s** (61.1%利用率)
- 相比nctigpu提升: **+32.8%**
- 最优TPR: **32** (适用于avgNnz=85矩阵)

---

## TPR优化结果

### TPR穷尽测试

| TPR | 时间(ms) | 带宽(GB/s) | 利用率 | 每warp行数 |
|-----|----------|------------|--------|-----------|
| 4 | 0.995 | 299 | 16.2% | 16 |
| 8 | 0.372 | 799 | 43.3% | 8 |
| 16 | 0.278 | 1072 | 58.2% | 4 |
| **32** | **0.264** | **1127** | **61.2%** | **2** ⭐ |
| 64 | 0.331 | 899 | 48.8% | 1 |

**最优配置**: TPR=32, blockSize=256

### TPR选择逻辑验证

```
avgNnz=85 → 选择TPR=32 → 正确!
```

## 性能对比

### 内核性能 (hcTracer验证)

| 指标 | nctigpu | 优化后 | 提升 |
|------|---------|--------|------|
| 内核时间 | 0.3565 ms | 0.2684 ms | **+32.8%** |
| 内核带宽 | 836 GB/s | 1111 GB/s | **+32.8%** |

| 模式 | 时间 | 说明 |

## 物理限制说明

| 参数 | 值 |
|------|-----|
| Mars X201峰值带宽 | 1843.2 GB/s |
| 当前利用率 | **61.1%** |
| 优化后带宽 | 1111 GB/s |

**说明**: 3倍内核性能(2520GB/s)会超过峰值带宽，物理上不可能。

## 建议使用方式

### 单次执行
```cpp
spmv_fp32_execute_direct(numRows, nnz, d_rowPtr, d_colIdx, d_values, d_x, d_y, 0);
```

### 迭代算法 (推荐)
```cpp
// 预处理：矩阵加载到GPU
spmv_fp32_matrix_handle_t handle;
spmv_fp32_create_matrix_device(&handle, ...);

// 每次迭代只需传输x/y向量
for (int iter = 0; iter < maxIter; iter++) {
    spmv_fp32_execute_device(handle, d_x, d_y, ...);
}
```
